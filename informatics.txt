-- –ö–æ–¥–∏—Ä–æ–≤–∫–∞ —Ç–µ–∫—Å—Ç–∞:

ASCII (American Standard Code for Information Interchange)
= (old, basic) character encoding standard.
- It uses 7 bits (128 characters).
- It contains English characters, digits, punctuation (example: [A-Z],[a-z],[.!?\n\t,(etc)]).
example: symbol "A" = 65(dec) = 1000001(binary)

Unicode = Universal character encoding standard
p.s. unicode is not encoding (–Ω–µ –∫–æ–¥–∏—Ä–æ–≤–∫–∞). It's a table (—Ç–∞–±–ª–∏—Ü–∞).
- –¥–∞–≤–Ω–æ: different encoding tables for different languages. –ó–∞—Ç–µ–º –ø—Ä–∏–¥—É–º–∞–ª–∏ –µ–¥–∏–Ω—É—é —Ç–∞–±–ª–∏—Ü—É Unicode.
(Unicode table contains all languages, digits, punctuation, graphical symbols, etc)
- "Code point" = code for symbol. Symbol allocates: from 1 to 4 bytes.
example: letter A = U+0041; smiley emoji üòÄ = U+1F600.

UTF-8 (Unicode Transformation Format - 8-bit)
