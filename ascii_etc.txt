

ASCII (American Standard Code for Information Interchange)
= (old, basic) character encoding standard.
- It uses 7 bits (128 characters).
- It contains English characters, digits, punctuation (example: [A-Z],[a-z],[.!?\n\t,(etc)]).
example: symbol "A" = 65(dec) = 1000001(binary)
